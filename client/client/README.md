# ShopSphere 🛒

A full-stack MERN E-Commerce Platform built using MongoDB, Express.js, React.js, and Node.js.
## Live Links

Frontend: https://shopsphere-beta-eight.vercel.app

Backend: https://shopsphere-backend-zlug.onrender.com

GitHub: https://github.com/shaikabdulsamad832-gif/Shopsphere
## Features

### User Features

* User Registration & Login
* Product Browsing
* Product Search
* Product Filtering
* Product Details Page
* Wishlist Management
* Shopping Cart
* Coupon System
* Checkout Process
* Razorpay Payment Integration
* Order Tracking
* User Profile
* Dark Mode

### Admin Features

* Hidden Admin Login
* Admin Dashboard
* Product Management
* Add Product
* Edit Product
* Delete Product
* Order Management
* Revenue Analytics

## Tech Stack

### Frontend

* React.js
* React Router
* Axios
* CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas

### Payment Gateway

* Razorpay

```bash

npm install

```
